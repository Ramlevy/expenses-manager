Expenses Manager Application
Mobile Course 2016

This is an application that tracks your expenses
and shows the balance of the money that was spent so far.
Besides, it shows in which categories the money was spent.
In the settings screen you can set your frequency (daily, weekly, monthly) expenses limit.
This application was made by
Mor Shitrit and Ram Levy.
